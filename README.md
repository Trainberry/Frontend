# Trainberry - Frontend

This is the frontend to manage your server and trains!

## How to run

* Install deps with `npm install`.
* Set the server address (without the final `/`) in `PUBLIC_SERVER_URL` environnement variable.
* Then, simply run `npm run dev`.
